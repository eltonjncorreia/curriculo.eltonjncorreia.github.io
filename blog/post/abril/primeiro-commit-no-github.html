Enviando o primeiro arquivo para o github

28 de Dezembro de 2017

Olá vamos aprender a usar o básico e também estaremos enviado arquivo pelo terminal para o github.
Tentarei explicar da maneira mais fácil.
Crie uma conta no https://github.com/ logo depois criaremos um repositório.
Para demostrar siga os passos:

1º passo :

Após ter logado, procure pela a sinal de mais no topo “ + create new repository”.
Na próxima tela defina o nome do seu repositório exemplo “teste”.
As configurações deixe como estão. Logo clique na opção “create repository”.
Na próxima pagina ficaram as instruções de utilização, não faça nada apenas anote a linha que contém o conteúdo semelhante a descrita abaixo: https://github.com/seu-usuario/teste.git precisaremos dessa linha depois.
Agora devemos abrir o terminal onde a brincadeira acontece.

2º passo:

Faremos uma configuração que não tinhamos realizado antes,
É bem simples, é recomendado que se faça a adição do nome de usúario e o e-mail que foi usado na conta do github, essa configuração será feita apenas uma vez caso você passe o comando “ --global” pois sempre vai ser usada esta informação em qualquer coisa que faça nesse sistema.
Caso deseje usar outra conta de e-mail em projetos diferentes é possível através do comando sem a opção “--global”

Entre no diretório:

cd “/home/seu_usuario/”

abra o terminal e digite:

$ git config --global user.email "email-que-usou-pra-criar-a-conta-do-github"
$ git config --global user.name "nome-do-seu-usuario-de-login-do-github"

Agora iremos criar um diretório onde ficará os projetos git, caso já tenha criado entre nele.

Crie um diretorio com o comando :

mkdir nomeDoDiretorio-git
cd nomeDoDiretorio-git

Agora criaremos um diretório para alocar o repositório que tínhamos criado anteriormente no site.
dentro desse diretório crie um arquivo chamado teste-git.

3º passo

Agora começaremos a utilizar o Git para gerir o repositório. Para inicializar as configurações, digite no terminal:

git init

Adicione mudancas ao arquivo que criamos antes, chamado teste-git.
faça: echo “texto que quero introduzir em arquivo” >> teste-git

Para adicionar: git add teste

O mais importante sempre que adicionar algo com "git add", terá que criar um "commit". Dessa forma a alteração será identificada pelo git.
Digite o comando: git commit -m “commitando arquivos no git ”
Aquele endereço que pedi pra anotar, vamos usá-lo agora:

git remote add origin https://github.com/seu-usuario/teste.git git status

Para enviar ao repositório github: git push -u origin master

Nessa parte pedirá o nome do usuário que usa pra logar no GitHub e a senha. Recarregue a página do github no navegador, e veja se aconteceu as alterações. Toda vez que adicionar ou alterar algo inclua ao git é uma boa prática: git status Ele irá listar o que foi adicionado ou alterado. Depois de feito essa configurações iniciais, bastará apenas usar os seguintes comandos para administrar:

git status
git add seu-arquivo
git commit -m “mais uma”
git push -u origin master
Caso crie outro repositório. Terá que repetir as configurações no início do tutorial.
